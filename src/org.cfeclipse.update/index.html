CFEclipse Update${site.suffix} Site

This is the update site for the CFEclipse Eclipse plugin. You need to have Eclipse version 4.2 or above, and add an update site using this URL:

http://cfeclipse.org/update${site.suffix}/

to install it.

Installation instructions are here: https://github.com/cfeclipse/cfeclipse/wiki/Installing-CFEclipse

Alternatively, theoretically, you can download an archived update site from here: http://cfeclipse.org/update-dev/cfeupdatesite.zip
